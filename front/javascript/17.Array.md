* 清空数组

```
arr.length = 0
```

* splice

```
arr = [1, 2, 3, 4, 5];
// 删除
arr.splice(2, 1); // [1, 2, 4, 5]

// 插入
arr.splice(2, 0, 3); // [1, 2, 3, 4, 5]

// 替换
arr.splice(2, 1, 9); // [1, 2, 9, 4, 5]
```

* join

```
arr = [1, 2, 3, 4];
arr.join('-'); // "1-2-3-4"
```

* sort
```
arr = [1, 11, 2, 13, 4, 15];

arr.sort(); // [1, 11, 13, 15, 2, 4]

arr.sort((n1, n2) => {
	return n1 - n2;
});    // [1, 2, 4, 11, 13, 15]
```
