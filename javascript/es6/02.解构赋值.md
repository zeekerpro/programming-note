### 解构赋值
+ 声明和赋值必须在一个表达式中
+ 赋值表达式左右两边解构必须一致 : 数组，json
+ 右边必须是一个合法的表达式

``` js
let arr = [1, 2, 3];
let [a, b, c] = arr;
```

json
```
let obj = {a : 1, b : 2, c : 3};
let {a, b, c} = obj;
```

array
```
let obj = [{a : 1, b : 2, c : 3}, [4, 5, 6], 90, "str"];
let [json, arr, num, string] = obj;
let [{a, b, c}, [n1, n2, n3], nu, st] = obj;
```

```
let [a, b]; 
[a, b] = [1, 2] //不合法，解构赋值必须在一个表达式中
```

```
let [a, b] = {1, 2} //不合法，右边必须是一个合法的表达式
```
