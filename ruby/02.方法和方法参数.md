### 方法的分类
- 实例方法


- 类方法
	接收者不是实例对象而是类对象本身，其实类本身是 Class 类的实例。类方法就是在这个 Class 类的实例上添加的方法。
	可以使用 :: 代替 . 来调用类方法


- 函数式方法
	没有接收者的方法就是函数式方法, 如 ：
	print "hello"
	sleep 10


### 方法参数

- 块

- 参数个数不确定
	```
	def foo(*args)
		args	
	end

	p foo(1, 2, 3) #=> [1, 2, 3]
	```
	表示多个参数合并成一个数组 args

- 参数解构
	```
	arr = [2, 3]
	def foo(a, b, c)
		a + b + c
	end

	p foo(1, *arr) #=> 6
	```
	*arr 表示将数组解构成对应的参数, 这种用法需要参数个数和解构后的数组个数一致

- 关键字传参
	```
	def foo(x: 0, y: 0, z: 0)
		x * y * z
	end

	foo(z: 8, y: 1) #=> 0 

	#  可以使用散列传参
	args = {x: 2, y: 3, z: 4}
	foo(args)  #=> 24
	```
	参数顺序可以不固定

- 默认参数
	```
	def foo(a = 90)
		a
	end

	foo   #=> 90
	foo(80) #=> 80
	```
