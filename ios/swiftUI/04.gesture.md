###  拖动

1. 设置位置状态	
``` swift
@State var positionState = CGSize.zero
```

2. 添加事件处理
```
.gesture(
	DragGesture()
		.onChanged{ value in
			self.positionState = value.translation
		}
		.onEnded{ value in
			self.positionState = CGSize.zero
		}
)
```

3. 使用 位置状态变量 positionState 实现拖动效果
```
.offset(x: positionState.width, y: positionState.height)
```
